## 设计小型控制器

<Figure src="设计小型控制器.svg" float >
    设计小型控制器题图
</Figure>

RA、RB、RC 均初始化为 0，如果 OF=0 则将运算结果先打入 RC 再打入 RB，OF 为溢出标志。运行时如伪码所示：

```
while(1){
    x->RA
    RA+RB
    if(!OF){
        RA+RB->RC
        RC->RB
}}
```

1. 画 ASM 图，状态从 `S0` 到 `Sn`。
2. 写出全部控制信号表达式。
3. 写出触发器次态方程、激励函数。
