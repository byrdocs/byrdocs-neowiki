## 四、计算题
（5分）如图所示电路，当$R_{L}=3\Omega$时获得最大功率3W，求$R$的大小和此时电流源供出的功率P。

<Solution>
    答：负载 $R_L$ 左边等效电阻为 $2 + R$，\quad（1 分）

    当 $2 + R = R_L = 3$ 时有最大功率传输，故 $R = 1\ \Omega$，\quad（1 分）

    由 $U_{OC} = R\,i_s$ 及最大功率公式 $\dfrac{(U_{OC})^2}{4} R_s = \dfrac{(R i_s)^2}{4} R_L = 3\ \text{W}$，得 $i_s = 6$，\quad（1 分）

    此时电流源右边纯电阻电路等效电阻为 $(2+3)\parallel 1 + 2 = \dfrac{17}{6}\ \Omega$，\quad（1 分）

    故功率 $P = i_s^2 \cdot \dfrac{17}{6} = 6^2 \cdot \dfrac{17}{6} = 102\ \text{W}$，\quad（1 分）
</Solution>

## 六、计算题
在如图所示的电路中，$t < 0$时电路处于稳态，$t = 0$时开关由1位切换到2位，求$t≥0$时的与电容连接的等效电阻，时间常数$τ$，$u_{C}(t)$的零输入响应，零状态响应和全响应。

<Solution>
    解：$u_C(0^+) = u_C(0^-) = -2/(1+2+2)*5 = -2\,V$ \quad（3 分）

    $u_C(\infty) = 2/(2+2)*10 = 5\,V$ \quad（3 分）

    等效电阻 $R = 2k\,\Omega \parallel 2k\,\Omega = 1k\,\Omega$ \quad（2 分）

    时间常数 $\tau = RC = 4*10^{-3}\,s$ \quad（2 分）

    零输入响应 $u_C(t)_{zir} = u_C(0^+) (e^{-\frac{t}{\tau}}) = -2(e^{-250t})\,V$ \quad（2 分）

    零状态响应 $u_C(t)_{zsr} = u_C(\infty)\,(1 - e^{-\frac{t}{\tau}}) = 5\,(1 - e^{-250t})\,V$ \quad（2 分）

    全响应 $u_C(t) = u_C(t)_{zir} + u_C(t)_{zsr} = (5 - 7e^{-250t})\,V$ \quad（1 分）
</Solution>

## 九、计算题
如图所示二端网络N，已知$u(t)=40\cos (100t+60°)V$，$i(t)=10\cos (100t+90°)$。请写出网络N的瞬时功率P(t)的表达式，求网络N吸收的平均功率，无功功率Q，视在功率S和功率因素$λ$。

<Solution>
    解：$P(t) = 40\cos(100t + 60^\circ) * 10\cos(100t + 90^\circ)$ \quad 3 分

    $P = UI\cos\varphi_{UI} = \frac{40}{\sqrt{2}} \frac{10}{\sqrt{2}} \cos(30^\circ) = 100\sqrt{3}\ \text{W}$ \quad 3 分

    $Q = UI\sin\varphi_{UI} = \frac{40}{\sqrt{2}} \frac{10}{\sqrt{2}} \sin(30^\circ) = 100\ \text{VAR}$ \quad 3 分

    $S = UI = \frac{40}{\sqrt{2}} \frac{10}{\sqrt{2}} = 200\ \text{VA}$ \quad 3 分

    $\lambda = \cos\varphi_{UI} = \cos(30^\circ) = \sqrt{3}/2$ \quad 3 分
</Solution>

## 十、计算题
如图所示电路，其中激励为交流电$u(t)=100\sqrt{17} \cos(t)V$，电流$i$与电压$u$同相时，电路中会发生什么现象？测得电感两端电压的有效值为20V，电路中电流的有效值为2A，求电路中元件$R$,$L$,$C$的值。

    $\omega L = 20/2 = 10,\ L = 10\,\text{H}$ \quad（2 分）

    $\omega = \frac{1}{\sqrt{LC}} = 1,\ C = 0.1\,\text{F}$ \quad（2 分）
</Solution>
