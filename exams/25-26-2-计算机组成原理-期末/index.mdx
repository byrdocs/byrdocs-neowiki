## 六
某计算机指令字长为 16 位，按字节编址。指令分为 R、I、B 三类，寄存器编号字段长度为 3 位，`opcode` 字段长度为 4 位，R 型指令的 `func` 字段长度为 3 位，imm 和 offset 是有符号整数。指令格式与功能如下：

| 类型 | 指令格式 | 功能说明 |
| --- | --- | --- |
| R 型 | opcode(4) rs(3) rd(3) rt(3) func(3) | R[rt] $\leftarrow$ R[rs] op R[rd] |
| I 型 | opcode(4) rs(3) rt(3) imm(6) | R[rt] $\leftarrow$ M[R[rs] + imm] |
| B 型 | opcode(4) rs(3) offset(9) | 若 R[rs] = 0，则 PC $\leftarrow$ PC + 2 + 2 $\times$ offset |

部分指令编码如下：

| 指令 | opcode | func |
| --- | --- | --- |
| add | 0000 | 001 |
| sub | 0000 | 000 |
| load | 待补 | 无 |
| beqz | 待补 | 无 |

回答下列问题：  
1. 该计算机最多可设置多少个通用寄存器？  
2. 已知一条机器指令为：0000 001 010 011 000，并给出相关寄存器初值。请按 R 型指令格式解析该指令，说明其对应的操作，并求执行后目标寄存器的值。
3. 某条 load 指令中，rs = 1，rt = 5，imm = -4。采用寄存器间接寻址，已知给出的地址处连续两个字节的内容为 AB 和 CD，且采用大端序存放。请计算有效地址，并求装入 R[5] 的数据
4. 某条 beqz 指令存放地址为 1000H，执行该指令时 PC = 2000H，目标寄存器为 R[3]，offset = -2。分别求当 R[3] = 0 和 R[3] = 1 时，指令执行后的 PC 值。
