### 1.（4 分）

具有五层协议的网络体系结构包括哪些层，各层的主要功能是什么？

<Solution>
物理层、数据链路层、网络层、传输层、应用层。

数据链路层：负责点到点的传输。

传输层：负责端到端的传输。

应用层：为用户提供应用接口。
</Solution>

### 2.（4 分）

在信噪比为 $30\,\mathrm{dB}$ 的信道上发送信号，最大数据率约是多少？

<Solution>
根据 $\left(\frac{S}{N}\right)_{db} = 10\log_{10}\frac{S}{N}$ 可以算出 $\frac{S}{N} = 1000$

然后根据香农公式可得：$W = B\log_2{\left(1+\frac{S}{N}\right)} = B\log_2{1001}$

由于题目中未给出 $B$，因此 $W$ 即为结果。
</Solution>

### 3.（4 分）

假定 $1\,\mathrm{km}$ 长的 CSMA/CD 网络的数据率为 $1\,\mathrm{Gbit/s}$。设信号在网络上的传播速率为 $20\times10^4\,\mathrm{km/s}$。求：

1. 发送站点的冲突检测时间是多长？

<Solution>
因为 $RTT = 2\tau$ 且 $2\tau = 2\times\frac{1km}{20\times10^4km/s} = 1\times 10^{-5}s$
</Solution>

2. 最短帧长是多少字节？

<Solution>
由于 $T_{trans} \geq RTT$ 可得，$\frac{W bit}{1\times 10^9 bit/s} \geq 1\times 10^{-5}$，解得 $W = 10^4 bit = 1250 byte$
</Solution>

### 6.（8 分）

假设 TCP 的拥塞窗口在置为 $32\,\mathrm{kB}$ 时发生了定时器超时。请计算拥塞门限值是多少？如果接下来的 7 次传输全部成功的话，接收窗口的大小始终为 $17\,\mathrm{kB}$，请计算每次成功传输后的发送窗口并完成下表。假设最大报文段长度为 $1\,\mathrm{kB}$。

| 传输次数 | 第1次 | 第2次 | 第3次 | 第4次 | 第5次 | 第6次 | 第7次 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 发送窗口大小（KB） |  |  |  |  |  |  |  |

<Solution>
Threshold 转变为 16KB。进入慢启动，cwnd 置为 1KB，rwnd 始终等于 17KB，发送窗口 = min(cwnd, rwnd)

第一次的 cwnd 是 1KB，第二次 2KB，第三次 4KB，第四次 8KB，第五次 16KB，此时达到阈值，进入拥塞避免阶段。

第六次的 cwnd 是 17KB，第七次的 cwnd 是 18KB，但是，发送窗口取 rwnd 和 cwnd 的较小者，因此发送窗口仍为 17KB。

</Solution>
