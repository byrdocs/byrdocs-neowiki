## 一、填空题（每小题4分，共40分）

1. 已知 $P(A) = P(B) = P(C) = \frac{1}{4}, \, P(AB) = P(BC) = 0, \, P(AC) = \frac{1}{8}$，求 $P(A \cup B \cup C)$。<Blank />

2. 判断题：若 $P(AB)=0$，那么 $AB=\varnothing$。<Blank />

3. 设随机变量 $X$ 与 $Y$ 相互独立，且 $X \sim B\left(1, \frac{1}{3}\right), \, Y \sim B\left(2, \frac{1}{2}\right)$，求 $P\{X=Y\}$。<Blank />

4. 袋中有50个乒乓球，其中20个是黄球，30个是白球。今有两人依次随机地从袋中各取一球，取后不放回，求第二人取得黄球的概率。<Blank />

5. 设随机变量 $X \sim N(\mu, \sigma^2), \, Y \sim N(\mu, \sigma^2)$，且设 $X, \, Y$ 相互独立，求 $Z_1 = \alpha X + \beta Y$ 和 $Z_2 = \alpha X - \beta Y$ 的相关系数（$\alpha$ 和 $\beta$ 是不为 0 的常数）。<Blank />

6. 设随机变量 $X$ 和 $Y$ 的数学期望都是 2，方差分别为 1 和 4，相关系数为0.5，试根据切比雪夫不等式给出 $P\{|X - Y| \geq 6\}$ 的估计。<Blank />
